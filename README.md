# Toast Machine

Toast Machine è un distributore self-service di software libero, scritto in
Python/GTK. Permette di masterizzare distribuzioni GNU/Linux su CD/DVD e di
copiare immagini ISO, raccolte di software libero e materiale informativo su
supporti USB e SD.

Il progetto è stato pensato per eventi pubblici come i Linux Day ed è
rilasciato sotto licenza GPLv2.
